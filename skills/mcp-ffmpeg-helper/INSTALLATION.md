## 🔧 功能特性

MCP FFmpeg Helper 提供以下强大的视频处理功能：

### 1. **视频信息获取**
- 获取视频文件的详细信息（时长、分辨率、编码等）
- 分析视频参数和元数据

### 2. **视频格式转换**
- 支持各种视频格式之间的转换
- 自定义输出格式和参数

### 3. **音频提取**
- 从视频中提取音频轨道
- 保存为各种音频格式

### 4. **视频创建**
- 从图像序列创建视频
- 制作延时摄影效果

### 5. **视频编辑**
- 裁剪视频（时长和区域）
- 为视频添加水印
- 调整视频参数

### 6. **音频处理**
- 裁剪音频文件
- 处理音频轨道

### 7. **帧提取**
- 从视频中提取帧为图像序列
- 生成视频缩略图

## 📋 支持的格式

### 视频格式
- 输入：MP4, AVI, MOV, MKV, WMV, FLV, etc.
- 输出：MP4, AVI, MOV, MKV, WebM, etc.

### 音频格式
- 输入：MP3, WAV, AAC, FLAC, etc.
- 输出：MP3, WAV, AAC, FLAC, etc.

### 图像格式
- 输入：JPG, PNG, BMP, GIF, etc.
- 输出：JPG, PNG, BMP, etc.
