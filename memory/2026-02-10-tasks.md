### 工具功能对比

#### MCP 图像提取器 (Image Extractor)
- **核心功能**: 数据提取和格式转换
- **适用场景**: AI 模型输入准备、图像内容分析、OCR文字提取、对象检测
- **技术栈**: TypeScript/Node.js
- **输出格式**: base64编码、文本信息、结构化数据

#### MCP 图像处理工具 (Image Processing Tool)
- **核心功能**: 图像编辑和批量处理
- **适用场景**: 专业图像编辑、大量图片优化、批量格式转换、滤镜效果
- **技术栈**: Python (Pillow + NumPy)
- **输出格式**: 处理后图像、图像集合
